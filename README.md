# Estructura de Datos - Implementación de Grafos en Java

# Descripción
Este proyecto contiene la implementación de grafos en Java utilizando dos representaciones diferentes:

- Matriz de adyacencia (grafo dirigido)
- Lista de adyacencia (grafo no dirigido)

Ambas implementaciones incluyen validaciones para evitar errores como duplicación de aristas o vértices inexistentes.

---

# Ejercicio 1 - Matriz de Adyacencia (Dirigido)

Se implementa un grafo dirigido utilizando una matriz bidimensional.
Cada posición [i][j] indica si existe una arista desde el vértice i hacia el vértice j.

Clases:
- GrafoMatriz.java
- MainMatriz.java

---

# Ejercicio 2 - Lista de Adyacencia (No Dirigido)

Se implementa un grafo no dirigido utilizando una estructura dinámica basada en listas.

Clases:
- GrafoLista.java
- MainLista.java

---

# ▶ Cómo ejecutar

Compilar y ejecutar las clases:

- matriz.MainMatriz
